忍術 — The Art of Stealth Efficiency

Classical ninjutsu reframed for agent-driven systems. Click to expand.

👁️‍🗨️
Concealment
隠形術 — Ongyo-jutsu
Minimize agent footprint. Fewer tokens per operation. Invisible efficiency.
Caching, batching (群影), lazy evaluation (眠刀), model stepping (影踏み), prompt compression (囁き). Every unnecessary token is a footprint the enemy can follow.
🎭
Adaptation
変装術 — Hensō-jutsu
One agent, many faces. Reduce headcount without reducing capability.
Role-shifting (変化), graceful degradation (遁術), foreign system blending (潜入), model stepping (影踏み). The ninja wears what the mission requires.
🔍
Intelligence
諜報術 — Chōhō-jutsu
Know where -1 is happening before it hemorrhages. Detection is cheaper than repair.
Five intelligence networks: Kanja monitors, spend sentinels, resistance scouts, cascade watchers, trend analysts. Information is the cheapest weapon (情報は最も安い武器).
💨
Escape Arts
遁術 — Ton-jutsu
Graceful degradation. Escape expensive operations without losing state.
Checkpoint frequently, reduce scope, transfer to lower-cost agents. Four degradation levels from full operation to survival mode. Never lose progress to a resource gate.
🔥
Fire Techniques
火術 — Ka-jutsu
Controlled burns. Strategic token investment with measurable ROI.
Decision tree: Is the burn controlled or waste? ROI requirements, compound value tests, the Seven Wastes (七つの無駄). Every flame must be justified by what it builds.
🌊
Water Techniques
水術 — Sui-jutsu
Flow around obstacles. Path B over Path A when the wall appears.
API blocked? Find another endpoint. Service down? Use cache. Auth wall? Cheapest auth method. Exponential backoff, never retry same approach twice. Water never fights the rock.
🥷
Shadow Agents
間者 — Kanja
Zero-token observers that detect -1 events before they compound.
Bash scripts, not AI. Cron monitors, not heartbeat polls. The cheapest agent consumes zero tokens — it's just a shell script watching the world and whispering alerts.
