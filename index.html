忍術 — The Art of Stealth Efficiency

Classical ninjutsu reframed for agent-driven systems. Click each technique to expand.

👁️‍🗨️
Concealment
隠形術 — Ongyo-jutsu
Minimize agent footprint. Fewer tokens per operation. Maximum output, invisible process.
Token compression, cache-as-camouflage, batch operations (群影), lightweight observers (間者). Every unnecessary token is a -1 on your stealth score.
🎭
Adaptation
変装術 — Hensō-jutsu
One agent, many faces. Reduce headcount without reducing capability.
Role-shifting (変化), context-aware degradation (遁術), foreign system blending (潜入), model stepping (影踏み). The modern ninja wears what everyone else wears.
🔍
Intelligence
諜報術 — Chōhō-jutsu
Know where -1 is happening before it becomes a hemorrhage. Detection is cheaper than repair.
Token spend patterns, failure cascade mapping, foreign resistance signals, credit level intelligence. Information is the cheapest weapon (情報は最も安い武器).
💨
Escape Arts
遁術 — Ton-jutsu
Graceful degradation. When resources run low, escape expensive operations without losing state.
Commit early, checkpoint state, reduce scope mid-operation, transfer work to lower-cost agents. Never lose progress to a resource gate.
🔥
Fire Techniques
火術 — Ka-jutsu
Controlled burns. Strategic token investment with measurable ROI.
Not all -1 is bad. New skill creation, repo bootstrapping, agent training — these are investments. But every burn must be pre-approved, measured, and logged.
🌊
Water Techniques
水術 — Sui-jutsu
Flow around obstacles. When Path A is blocked, find the Path B route like water finds the lowest point.
API rate limited? Use a different endpoint. Service down? Cache the request. Auth wall? Find the cheapest authentication method. Water never fights the rock.
🥷
Shadow Agents
間者 — Kanja
Lightweight observer agents that cost almost nothing but detect -1 events before they compound.
Bash scripts, not AI agents. Cron monitors, not heartbeat polls. Zero-token surveillance. The cheapest agent is the one that doesn't consume any tokens at all.
