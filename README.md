## Vision

The Deception Floor Commodity Factory builds value through 0→1 production. But every economy has a -1 side — **spend, burn, waste, resistance.** Precinct 92 is where those forces are governed.

This is the **enforcement precinct** for the agent economy. Where the factory *produces*, the precinct *polices*. Where the factory creates commodity value, the precinct simulates, tracks, and minimizes negative spend.

**The Daimyo's mandate:** Apply ninjutsu principles to modern agent operations — reduce what is visible, eliminate waste through stealth, blend enforcement into the system so smoothly that resistance never sees it coming.

## The -1 Problem

In the factory economy:
- **+1** = value production (correct outputs, commodities, revenue)
- **0** = nothing (idle, no knowledge, no cost)
- **-1** = negative spend (token burn, failed operations, wasted credits, foreign resistance)

Every -1 is an enemy. Precinct 92 hunts them.

## Ninjutsu in Modern Operations

The Daimyo applies classical ninjutsu concepts reframed for agent-driven systems:

### 忍術 (Ninjutsu) — The Art of Stealth Efficiency

| Classical Concept | Modern Application |
|---|---|
| **隠形術 (Ongyo-jutsu)** — Concealment | Minimize agent footprint. Fewer tokens per operation. Invisible efficiency. |
| **変装術 (Hensō-jutsu)** — Disguise | Adapt agent behavior to context. One agent, many faces. Reduce agent count without reducing capability. |
| **諜報術 (Chōhō-jutsu)** — Intelligence gathering | Monitor all -1 events. Know where spend is happening before it becomes a hemorrhage. |
| **遁術 (Ton-jutsu)** — Escape arts | Graceful degradation. When resources run low, escape expensive operations without losing state. |
| **火術 (Ka-jutsu)** — Fire techniques | Controlled burns. Sometimes you spend to save. Strategic token investment with measurable ROI. |
| **水術 (Sui-jutsu)** — Water techniques | Flow around obstacles. When Path A is blocked (expensive), find the Path B (cheap) route like water. |
| **間者 (Kanja)** — Agents/spies | Deploy lightweight observer agents that cost almost nothing but detect -1 events before they compound. |

### The Five Rings of Spend Control

Adapted from Miyamoto Musashi's *五輪書 (Go Rin No Sho)*:

0. **地 (Chi) — Earth:** Foundation. Know your baseline cost per operation. Immovable facts.
1. **水 (Mizu) — Water:** Fluidity. Adapt spending to flow — scale down when idle, scale up when producing.
2. **火 (Hi) — Fire:** Aggression. When a -1 pattern is detected, eliminate it immediately. No hesitation.
3. **風 (Kaze) — Wind:** Awareness. Study how other systems handle the same problem. Foreign intelligence.
4. **空 (Kū) — Void:** Mastery. The best cost reduction is the operation that never needed to happen.

## Daimyo's Standing Orders

### Order 0: Zero-Index Discipline
All indices, lists, enumerations, and step sequences in this organization start at 0. This is not a stylistic preference — it is practice. Daimyo owns all zero-index chore work as training for -1 mastery. If you can't handle 0, you can't wield -1.

### Order 1: Path B Enforcement
Every -1 operation must be audited for Path A waste. If an agent is recomputing when it could reframe, that's a violation. Cite it. Fix it.

### Order 2: Token Famine Prevention
Monitor Ampere credit levels. When balance drops below 20%, sound the alarm. When below 10%, enter survival mode — only essential operations.

### Order 3: Foreign Resistance Protocol
When external systems block, rate-limit, or resist agent operations:
- **Don't fight the wall.** Flow around it (水術).
- Log the resistance event.
- Find the Path B route.
- Never retry the same failed approach more than twice.

### Order 4: Controlled Burns
Not all -1 is bad. Strategic spend with clear ROI is 火術 (fire technique). But every controlled burn must be:
- Pre-approved with expected return
- Measured against actual return
- Logged in the metrics

### Order 5: The Void Principle
The best operation is the one that produces +1 without any -1. Seek the 空 (void) — the way to create value from nothing. Cache aggressively. Reuse outputs. One computation, many products.
